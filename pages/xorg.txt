=================
Xorg config
=================

http://www.freebsd.org/doc/zh_CN/books/handbook/x-config.html

从 Xorg 7.4 开始， 可以使用 HAL 自动检测键盘和鼠标。
Ports sysutils/hal 和 devel/dbus 将被作为 x11/xorg 所依赖的包安装进系统。
并且需要在 /etc/rc.conf 文件中启用:

    ::
  
    hald_enable="YES"
    dbus_enable="YES"
    
    sudo Xorg -configure
    sudo nano /root/xorg.conf.new
    sudo Xorg -config xorg.conf.new -retro
    sudo X -config /root/xorg.conf.new
    sudo rm /root/xorg.conf.new
    cp xorg.conf.new /etc/X11/xorg.conf
    
    find / -type f -name xorg.conf &>/tmp/output &
    cat /tmp/output
    Xorg :0 -configure
    /etc/init.d/gdm3 stop
    /etc/init.d/gdm3 start
    startx
